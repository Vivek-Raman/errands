# Errands

Native iOS app for viewing today's Apple Calendar events.

## Requirements

- Xcode 16 or newer
- iOS 17 or newer

## Run

Open `Errands.xcodeproj` in Xcode, select an iOS simulator or device, and run the `Errands` scheme.

The app requests read access to Calendar events, then shows the current day's visible events with pull-to-refresh support.
